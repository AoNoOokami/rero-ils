RERO ILS

{% if current_i18n.language == 'en' %}

The RERO ILS project aims at creating the future RERO Integrated Library System, using components of the Invenio 3 framework. This site provides a permanent, public demonstration instance of the software under development, with daily integration of new features.

The project will start in January 2018. During Oct-Nov 2017, an early ILS prototype with a small subset of features, is being built and made available here. This prototype, as the main project, are developed using Agile principles, namely Scrum, pair programming and continuous integration.

{% endif %} {% if current_i18n.language == 'fr' %}

Le projet RERO ILS a pour but de créer le futur système de gestion de blibliothèque de RERO basé sur le framework Invenio 3. Ce site propose une instance publique et permanente du logiciel en cours de développement, avec une intégration continue des nouvelles fonctionnalités.

Le projet débutera en janvier 2018. Au cours des mois d'oct-nov 2017, un premier prototype du logiciel intégrant un échantillon de fonctionnalités a été développé et est mis à disposition ici. Ce prototype, à l'instar du projet principal, a été développé selon des principes agiles, à savoir Scrum, en programmation par paires et en intégration continue.

{% endif %} {% if current_i18n.language == 'de' %}

Das Projekt RERO ILS zielt darauf ab, das zukünftige Bibliotheksverwaltungssystem von RERO auf der Basis des Invenio 3 Frameworks zu schaffen. Diese Website bietet eine permanente, öffentliche Demonstration der Software in Entwicklung, mit täglicher Integration neuer Funktionen.

Das Projekt wird im Januar 2018 starten. Im Okt-Nov 2017 wurde ein früher ILS-Prototyp mit einer kleinen Teilmenge von Funktionen gebaut, welcher hier zur Verfügung gestellt ist. Dieser Prototyp wurde wie das Hauptprojekt nach agilen Prinzipien, nämlich Scrum, in Paarprogrammierung und mit kontinuierlicher Integration, entwickelt.
